CUDA 有两种API，即 Runtime API 与 Driver API。

  • nvidia-smi : GPU 驱动版本型号 、 CUDA Driver API型号
  • nvcc -V : CUDA Runtime API。

nvidia-smi版本大于nvcc -V的版本不会有什么问题
